# InformatikProjektGasthaus
Funktioniert mit Java version 8.
Die ormlite-jdbc-6.1.jar datei muss heruntergeladen werden ormlite.com/releases/.
Auszuführen ist die Gasthaus main Datei.
Wenn das Projekt ausgeführt wurde muss es am Ende beendet werden (0 drücken bei der Auswahl) ansonsten kann es beim nochmaligen start zu einem Fehler kommen.
